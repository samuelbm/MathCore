MathCore

MathCore – a modular C/C++ mathematics toolkit providing foundational math operations, including matrices, linear algebra, and extendable math utilities. Free for personal use; commercial use requires a separate license.

Overview

MathCore is designed as a collection of independent, reusable modules:

matrix_c/ – Core C library for matrix operations (creation, addition, multiplication, inversion, transpose, norms).

matrix_cpp/ – C++ wrapper providing object-oriented access to the C core.

geometry_3d/ – Placeholder for future 3D geometry modules.

cryptography/ – Placeholder for future cryptography tools.

Each module can be built and tested independently, making MathCore flexible for different projects.

Features

Modular C and C++ libraries

Clean memory management for matrices

Unit-tested with Google Test

Easy extension for new math components

Cross-platform support via CMake
